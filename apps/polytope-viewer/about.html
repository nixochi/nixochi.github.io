Polytope Viewer

2025-01-20

technical notes

built with three.js for rendering, using orbit controls for interaction. the geometry is computed directly—no model files, just mathematical definitions of vertices and edges. this keeps the tool lightweight and lets you explore the pure mathematical structure.

the code is open. if you want to add more polytopes or change how they're displayed, the source is available. polytopes reveal their secrets slowly, and everyone sees something different in them.
